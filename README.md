# VPay

```
git clone https://github.com/palliums-developers/vpay.git
cd vpay
```

## Build

Build for Windows and Linux on Ubuntu 18.04 and for OSX on latest OSX.
A 'Dockerfile' file is provided to build Linux and Windows binaries.
